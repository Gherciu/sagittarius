Version 1.3.0 has ben released. What's new in 1.3.0 version...

## What's new in version 1.3.0

- Added a new util to transform provided argument to a property path array - [toPath](https://gherciu.github.io/sagittarius/docs/doc-api-sagittarius-to-path)
- Added a new util that checks if path is a direct property of provided argument - [has](https://gherciu.github.io/sagittarius/docs/doc-api-sagittarius-has)
- Added a new util that gets the value at path of provided argument - [get](https://gherciu.github.io/sagittarius/docs/doc-api-sagittarius-get)
- Added a new util that sets the value at path of provided argument - [set](https://gherciu.github.io/sagittarius/docs/doc-api-sagittarius-set)
